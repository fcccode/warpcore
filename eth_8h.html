void eth_rx ( struct w_engine *const  w,
struct netmap_ring *const  r 
)

Receive an Ethernet frame.

This is the lowest-level RX function, called for each new inbound frame from w_rx(). Dispatches the frame to either ip_rx() or arp_rx(), based on its EtherType.

The Ethernet frame to operate on is in the current netmap lot of the indicated RX ring.

wBackend engine.
rCurrently active netmap RX ring.
